 
# Personal Introduction Program

## Project Overview
This is a simple Python program that asks the user for their name, age, and hobby. 
It then displays a friendly welcome message.

## Objectives
- Learn how to use input()
- Store data in variables
- Display output using print()

## Setup Instructions
1. Install Python or use Pydroid.
2. Create file named personal_intro.py
3. Run the program.

## Code Structure
- input() is used to take user data
- Variables store name, age, hobby
- print() displays output
